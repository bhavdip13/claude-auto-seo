Phase 1 — Extract & Install

Unzip the file and install the required software

⏱ 5 min
1

Extract the ZIP File

Right-click the downloaded claude-auto-seo.zip file and extract it to a folder you can find easily.

💡
Windows: Right-click the ZIP → "Extract All" → choose C:\Projects\claude-auto-seo → click Extract.
Mac: Double-click the ZIP file — it extracts automatically next to it.
Linux: Right-click → "Extract Here".
After extracting, open the folder and check you can see these files:
  • README.md (documentation file)
  • install.sh (installer for Mac/Linux)
  • install.ps1 (installer for Windows)
  • scripts/ (folder with all the automation)
  • config/ (folder for your settings)
  • .env.example (template for credentials)
2

Check if Python is Installed

Python is the engine that runs all automation scripts. Check if you already have it.

Required

Open your Terminal and type this, then press Enter:

Type this command exactly
python3 --version
✅
If you see Python 3.10 or higher (like Python 3.11, 3.12) — you're good! Skip to Step 3.
⚠️
If you see "command not found" or Python 3.9 or older — download and install Python from python.org/downloads.
Windows users: During install, make sure to check the box that says "Add Python to PATH" — this is very important!
3

Install Claude Code

Claude Code is the AI brain that writes blog posts and runs all SEO commands. Install it like any normal program.

Required
  1. Go to claude.ai/claude-code
  2. Download the installer for your operating system
  3. Run the installer just like any normal program installation
  4. After installing, test it by opening a new terminal and typing:
claude --version
You should see something like: claude 1.x.x — that means it's installed correctly.
4

Open Terminal Inside the Project Folder

All commands must be run from inside the claude-auto-seo folder. Here is how to do that.

💡
Windows easiest way: Open File Explorer → go into the extracted claude-auto-seo folder → click on the address bar at the top of the window → type cmd → press Enter. A Command Prompt opens right in that folder.

Mac/Linux: Open Terminal → type cd (with a space after it) → drag the claude-auto-seo folder into the Terminal window → press Enter.
Or type the path manually (change to match your actual folder location)
# Windows example:
cd C:\Users\YourName\Downloads\claude-auto-seo

# Mac example:
cd /Users/YourName/Downloads/claude-auto-seo

# Linux example:
cd /home/yourname/Downloads/claude-auto-seo
5

Run the Installer Script

This sets up Claude Code commands and then installs all required Python libraries.

Mac or Linux — run this first
chmod +x install.sh
./install.sh
Windows — run this in PowerShell (search "PowerShell" in Start menu)
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\install.ps1
All operating systems — then run this to install Python packages
pip install -r requirements.txt
⚠️
If pip is not found, try pip3 install -r requirements.txt instead.
Installation is successful when you see no red ERROR lines. Yellow "WARNING" lines are normal and fine to ignore.

⏱ 2 min

Run the Auto-Configurator

Give it your domain name and it does everything — scans your site, detects your business type, extracts your contact details, reads your WordPress posts, and creates all configuration files.

Required
Basic version — just your domain name (always works)
python3 scripts/auto_configure.py --domain yoursite.com
Better version — include your WordPress login for richer results
python3 scripts/auto_configure.py \
  --domain yoursite.com \
  --wp-user your_wordpress_username \
  --wp-pass "your app password"
💡

After running you will see these files auto-created:
  • config/site.json — your site URL, niche type, business settings
  • config/business.json — your email, phone, social profiles (for CTAs in every post)
  • config/keywords.md — keyword clusters built for your industry
  • context/brand-voice.md — writing style guide for your brand
  • context/internal-links-map.md — all your WordPress pages mapped
  • topics/queue.txt — 30 topics loaded and ready to write
  • .env — credentials template pre-filled with your URL and brand colors

Add Your Anthropic API Key

This is what lets the system write your daily blog posts automatically using Claude AI.

Required
  1. Go to console.anthropic.com/api-keys
  2. Sign up for a free account (or log in if you have one)
  3. Click the "Create Key" button
  4. Give it a name like Claude Auto SEO
  5. Copy the long key that appears — it starts with sk-ant-api03-
  6. Paste it into your .env file replacing your_key_here
Add this line in your .env file
ANTHROPIC_API_KEY=sk-ant-api03-your-actual-key-here
10

Add WordPress App Password

This lets the system publish blog posts and fix SEO issues on your site. You must create a special "App Password" — this is different from your normal WordPress login password.

Required
  1. Log in to your WordPress Admin at yourdomain.com/wp-admin
  2. In the left menu, click Users → then click on your username
  3. Scroll down until you see a section called "Application Passwords"
  4. In the box that says "New Application Password Name", type: Claude Auto SEO
  5. Click the button "Add New Application Password"
  6. WordPress shows you a password that looks like this: AbCd EfGh IjKl MnOp QrSt UvWx
  7. Copy it immediately — WordPress will never show it again!
  8. Paste it into your .env file
🚫
Do NOT use your normal WordPress login password here. You must create an App Password specifically. The App Password has spaces in it — keep the spaces.
Add these 3 lines to your .env file
WP_URL=https://yoursite.com
WP_USERNAME=your_wordpress_username
WP_APP_PASSWORD=AbCd EfGh IjKl MnOp QrSt UvWx

Add Gmail Credentials for Daily Reports

Every evening at 8 PM the system will email you showing everything it did that day, with links to verify each post. This needs a special Gmail App Password.

Required
  1. First, make sure Gmail 2-Step Verification is ON: go to myaccount.google.com/security → scroll to "2-Step Verification" → turn it on if not already
  2. Then go to myaccount.google.com/apppasswords
  3. In the "App name" field type: Claude Auto SEO
  4. Click "Create"
  5. Google shows a 16-character password like: abcd efgh ijkl mnop
  6. Copy it and paste into your .env file
Add these 3 lines to your .env file
SMTP_USER=your@gmail.com
SMTP_PASS=abcd efgh ijkl mnop
NOTIFICATION_EMAIL=you@yoursite.com
💡
NOTIFICATION_EMAIL is where the daily report gets sent. You can use the same Gmail or any other email address.

Do I Need My Laptop On 24/7?

The answer is NO — use GitHub Actions. It is free and runs on GitHub's computers.

No, you do NOT need your laptop running 24/7. GitHub Actions is a free service that runs your code on GitHub's servers. Your computer can be switched off. The system still runs at exactly the right times every day.

Python-specific hosting. Beginner friendly. Free tier runs 1 scheduled task per day.

35

Set Up GitHub Actions Step by Step

Follow these steps. Takes about 10 minutes and then your laptop can stay off forever.

  1. Create a free account at github.com
  2. Click "New repository" → name it claude-auto-seo → set it to Private
  3. Upload all your project files into the repository (do NOT upload the .env file)
  4. Go to your repository → click Settings tab at the top
  5. In the left sidebar click Secrets and variables → then Actions
  6. Click "New repository secret" for every line in your .env file — add each key and value one by one
  7. Now copy the workflow file to the right location:
# Create the GitHub Actions folder
mkdir -p .github/workflows

# Copy the pre-built workflow file
cp scripts/github-actions-template.yml .github/workflows/automation.yml
🔒
NEVER upload your .env file to GitHub. It contains your passwords. Use GitHub Secrets instead (Settings → Secrets → Actions → add each key from .env separately).
After uploading the workflow file: go to your GitHub repository → click the Actions tab at the top → you will see the automation workflow listed and running on a schedule.
