# /copywriting — Expert Copywriting

Write persuasive, conversion-focused copy for any purpose.

**Framework:** AIDA (Attention → Interest → Desire → Action)

When invoked with `/copywriting [context]`, produce:
- 3 headline variations (power word formula)
- Body copy with benefit-focused language
- CTA with urgency or value reinforcement
- Objection-handling language

Rules: Benefits > features. Specific > vague. Active > passive.

# /page-cro — Landing Page CRO Analysis

Analyze any page for conversion rate optimization opportunities.

**Audit framework:**
1. Above-fold: headline clarity, value prop, CTA visibility (5-second test)
2. Trust signals: testimonials, logos, guarantees, numbers
3. CTA optimization: copy, color, placement, frequency
4. Friction audit: form fields, steps to convert, cognitive load
5. Social proof: specificity, recency, relevance
6. Urgency/scarcity: authentic or manipulative?

Returns: CRO score (0-100), priority fixes, A/B test ideas.
